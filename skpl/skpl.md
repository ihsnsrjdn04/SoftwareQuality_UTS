# SKPL - Sistem CBT Online

## 1. Pendahuluan
Dokumen ini menjelaskan kebutuhan perangkat lunak untuk sistem CBT (Computer Based Test). Sistem ini digunakan untuk membantu proses ujian online secara terstruktur dan otomatis.

---

## 2. Tujuan Sistem
- Mempermudah pelaksanaan ujian berbasis komputer  
- Mengelola data peserta dan soal  
- Menyediakan hasil nilai secara otomatis  

---

## 3. Lingkup Sistem
Sistem digunakan oleh:

- Mahasiswa → mengikuti ujian  
- Admin → mengelola soal dan peserta  

---

## 4. Fitur Utama
- Login dan Registrasi  
- Ujian CBT  
- Penilaian otomatis  
- Ranking peserta  
- Admin (kelola soal)  

---

## 5. Kebutuhan Fungsional
- Sistem dapat melakukan login user  
- Sistem dapat memulai ujian dengan token  
- Sistem dapat menyimpan jawaban  
- Sistem dapat menghitung nilai otomatis  
- Sistem dapat menampilkan hasil  

---

## 6. Kebutuhan Non-Fungsional
- Sistem mudah digunakan  
- Respon cepat  
- Validasi input  

---

## 7. Struktur Data (Ringkasan)
Tabel utama:
- user  
- soal  
- jawaban  
- nilai  

---

## 8. Teknologi
- HTML  
- CSS  
- JavaScript  
- LocalStorage
