## Deskripsi

Repository ini digunakan untuk memenuhi tugas Ujian Tengah Semester(UTS) mata kuliah Software Quality.

---

## Tujuan

Tujuan dari pembuatan repository ini adalah:
- Mempelajari konsep Software Quality  
- Melakukan review software  
- Melakukan pengujian software  
- Mendokumentasikan hasil pengujian  

---

## Tools yang Digunakan
- GitHub (Version Control & Dokumentasi)
- Visual Studio Code
- Browser

---

## Metode Pengujian
Dalam pengujian sistem ini digunakan beberapa metode, yaitu:
- Black Box Testing (EP, BVA, Use Case)
- White Box Testing
- Grey Box Testing

---

## Struktur Repository
- docs → Dokumentasi alur pengerjaan & penggunaan GitHub  
- review → Walkthrough, Inspection, Technical Review  
- skpl → Spesifikasi Kebutuhan Perangkat Lunak  
- src → Source code sistem  
- testing → Pengujian dan hasil testing
